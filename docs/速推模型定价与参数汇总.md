## 视频生成模型

### Seedance 2.0 全能 (`st-ai/super-seed2`)
- **价格**: Fast 30积分/秒, 标准 50积分/秒 (文生); 含视频素材翻倍
- **参数**: model(`seedance_2.0_fast`), prompt, **functionMode**(`omni_reference`/`first_last_frames`), **ratio**(非aspect_ratio), duration(4-15 int), **filePaths**(首尾帧), **image_files**(参考图), **video_files**, **audio_files**

### Super Seed 2 Lite (`st-ai/super-seed2-lite`)
- **价格**: Fast 50积分/秒起
- **参数**: model(`seedance2.0_fast_direct`), prompt, duration(5/10/15), **ratio**, **image_files**, **video_url**

### Sora 2 (`fal-ai/sora-2/text-to-video` / `image-to-video`)
- **价格**: 40积分/秒 (160积分/4秒起)
- **参数**: prompt, duration(**int**, 仅4/8/12/16/20), aspect_ratio, resolution, **model**(`"sora-2"`内部参数), character_ids
- **注意**: duration 是整数, 只能是 4/8/12/16/20

### Seedance 1.5 Pro (`fal-ai/bytedance/seedance/v1.5/pro/text-to-video` / `image-to-video`)
- **价格**: 10-47积分/秒 (按分辨率)
- **参数**: prompt, aspect_ratio, resolution, duration(**字符串** "5"), **generate_audio**, camera_fixed, seed

### Hailuo 2.3 Pro (`fal-ai/minimax/hailuo-2.3/pro/text-to-video` / `image-to-video`)
- **价格**: 196积分/次(固定)
- **参数**: prompt, image_url(i2v), prompt_optimizer
- **⚠️ Pro 无 duration 参数**

### Hailuo 2.3 Standard (`fal-ai/minimax/hailuo-2.3/standard/text-to-video` / `image-to-video`)
- **价格**: 112积分(6秒), 224积分(10秒)
- **参数**: prompt, duration(**字符串** "6"), prompt_optimizer, image_url(i2v)

### Vidu Q3 (`fal-ai/vidu/q3/text-to-video` / `image-to-video`)
- **价格**: 28-62积分/秒 (按分辨率)
- **参数**: prompt, duration(**int**), aspect_ratio, resolution, **audio**(bool)

### Veo 3.1 (`fal-ai/veo3.1`)
- **价格**: 80-240积分/秒 (按分辨率+音频)
- **参数**: prompt, aspect_ratio, duration(**字符串带s**: "4s"/"6s"/"8s"), resolution("720p"/"1080p"/"4k"), **generate_audio**, negative_prompt, seed
- **⚠️ 不区分 t2v/i2v, 单一 ID**

### Kling O3 Pro (`fal-ai/kling-video/o3/pro/text-to-video` / `image-to-video`)
- **价格**: 89.6-112积分/秒
- **参数**: prompt, duration(**字符串** "5", 3-15秒), aspect_ratio, **generate_audio**

### Grok Video (`xai/grok-imagine-video/text-to-video` / `image-to-video`)
- **价格**: 20积分/秒
- **参数**: prompt, duration(**int**, 1-15), aspect_ratio, resolution

### 即梦视频 (`jimeng-video-3.5-pro` / `-10s` / `-12s`)
- **价格**: 10/20/30积分/秒 (按模型)
- **参数**: prompt, image_url, end_image_url
- **⚠️ 无 duration/aspect_ratio 参数**

### Wan 2.7 (`wan/v2.7/text-to-video` / `image-to-video`)
- **价格**: 40积分/秒
- **参数**: prompt, **audio_url**, aspect_ratio, resolution, duration(**int** 2-15), negative_prompt

### Wan 2.6 (`wan/v2.6/text-to-video` / `image-to-video`)
- **参数**: prompt, image_url(i2v), aspect_ratio, duration(**字符串**), resolution

## ⚠️ 代码参数对照注意事项

1. **duration 类型不统一**: Sora/Wan/Vidu/Grok 用 int; Seedance1.5/Kling/Hailuo Standard 用 string; Veo 用 "Xs" 格式
2. **Hailuo Pro 无 duration**: 只有 Standard 版有 duration 参数
3. **即梦视频无 duration/aspect_ratio**: 只支持 prompt + image_url + end_image_url
4. **Veo 3.1 单一ID**: 不区分 text-to-video/image-to-video
5. **Super Seed2 用 ratio 不用 aspect_ratio**: 且有 functionMode/filePaths 等特殊参数
6. **Gemini 用 size 不用 image_size**: 且用 image_urls 数组
7. **Sora 有内部 model 参数**: params 里还需 model: "sora-2"
8. **Kling 用 generate_audio 不用 audio**: 与 Vidu 的 audio 不同
