## 参数传递策略

### 1. 特殊模型
对于有特殊参数要求的模型（如 veo-3.1 的 duration 格式），我们：
1. 正确处理特殊格式
2. 保留用户传入的其他参数
3. 不丢失任何参数

### 2. 通用模型
对于没有特殊要求的模型：
1. 保留所有用户传入的参数
2. 自动处理 `image_url`/`aspect_ratio`（根据是否有图片）
3. 不自动添加 `duration`（让上游 API 处理）

## 检查清单

### ✅ 已检查的参数
- `model`: 所有模型
- `prompt`: 所有模型
- `image_url`: 所有支持图生图/图生视频的模型

### ⚠️ 可能需要检查的参数（如果上游 API 支持）
- `guidance_scale`: 引导强度
